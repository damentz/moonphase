Troubleshooting
---------------

Users on certain versions of Windows have reported seeing a dialog box with the following text:

"The program can't start because MSVCR100.dll is missing from your computer.  Try reinstalling the program to fix this problem."

If you see this, you will need to download and install an update from Microsoft via the following links:

32bit Windows (x86) - http://www.microsoft.com/en-us/download/details.aspx?id=8328
64bit Windows (x64) - http://www.microsoft.com/en-us/download/details.aspx?id=13523
